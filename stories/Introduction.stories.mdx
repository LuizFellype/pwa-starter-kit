# PWA Starter Kit - Storybook

Welcome to the PWA Starter Kit component library! This Storybook contains all the themed components, interactive elements, and utilities that make up our comprehensive starter kit.

## Features

- **PWA Ready**: Progressive Web App capabilities with offline support
- **Themed Components**: Complete component library with light/dark theme support
- **Interactive Elements**: Filterable lists, search inputs, and multi-select components
- **TypeScript**: Full TypeScript support with proper type definitions
- **Accessibility**: WCAG compliant components with proper ARIA attributes

## Getting Started

### Theme Support

All components support both light and dark themes. Use the theme switcher in the toolbar to preview components in different themes.

### Component Categories

- **UI Components**: Basic building blocks like buttons, cards, and containers
- **Text Components**: Typography variants for different content types
- **Interactive Components**: Advanced components with filtering and search capabilities
- **Layout Components**: Containers and layout utilities
- **Loading States**: Skeleton components for loading states

### Usage

Each component story includes:
- **Controls**: Interactive controls to modify component props
- **Docs**: Comprehensive documentation with examples
- **Source Code**: View and copy the component source code
- **Accessibility**: Information about accessibility features

## Design System

Our design system follows these principles:

- **Consistency**: Unified visual language across all components
- **Accessibility**: WCAG AA compliant with proper contrast ratios
- **Flexibility**: Customizable through CSS variables and Tailwind classes
- **Performance**: Optimized for fast loading and smooth interactions

## Development

To run Storybook locally:

\`\`\`bash
npm run storybook
\`\`\`

To build Storybook for production:

\`\`\`bash
npm run build-storybook
\`\`\`
